# Miscellaneous Python Projects

This folder contains small, fun, and useful Python projects, showcasing general programming skills.


## Projects

### Password Generator
- **Description:** Generates random passwords of customizable length.  
- **Features:**
  - Choose password length (default: 12).  
  - Choose between strong (letters, numbers, symbols) or weak (letters, numbers).  


### To-Do CLI App
- **File:** `todo_cli.py`
- **Description:** Simple command-line to-do manager. Stores tasks in `todos.json` inside the `miscellaneous` folder.
- **Usage examples:**
  - `python todo_cli.py add "Buy milk"`
  - `python todo_cli.py list`
  - `python todo_cli.py list -a`
  - `python todo_cli.py done 1`
  - `python todo_cli.py remove 1`
  - `python todo_cli.py clear --yes`
